# Card artwork (Heroes & Wizards)

Card image files are **not included** in this repository. Add your own card artwork here so the app can serve them at `/cards/...`.

Required filenames and paths are defined in **`src/data/cards.ts`** (e.g. `quest_card.png`, `wizard_healer.png`, `heroes_wizards_card_back.png`, and event/hero card filenames). Place matching PNG (or other) files in this folder so the game can load them.
