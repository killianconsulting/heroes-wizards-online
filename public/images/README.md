# Site artwork (Heroes & Wizards)

Copy your original artwork files into this folder using these **exact** filenames so the site can use them:

| Filename                    | Where it’s used |
|-----------------------------|-----------------|
| `hw_logo.png`               | Game header and start screen title |
| `hw_horizontal_divider.png` | Horizontal divider above footer content |

Optional (for future use):

| Filename             | Use |
|----------------------|-----|
| `hw_background.jpeg` | Stone wall with moss — e.g. footer or section background |

- **Logo:** If `hw_logo.png` is missing, the site shows the text “Heroes & Wizards” instead.
- **Divider:** If `hw_horizontal_divider.png` is missing, the footer simply omits the divider image.
